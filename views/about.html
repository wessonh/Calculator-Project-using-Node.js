About My Calculator App

Welcome to my calculator web application. It is capable of these functions:

  • Basic arithmetic operations: addition, subtraction, multiplication, and division.
  • Handling of parentheses for complex calculations.
  • Support for decimals in numeric input.
  • Toggle functionality for switching between negative and positive numbers.
  • Clearing the current total and the input.

The user interface is designed with the following features:

  • Interactive buttons with hover and click states.
  • A scrollable history section displaying past calculations.
  • Option to clear the history for better organization.

Enjoy using my calculator app!
